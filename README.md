一个用于湖南财政经济学院（HUFE）校园网认证系统（eportal）自动登录的 Python 脚本
通过模拟 HTTP 请求实现自动登录校园网，适用于路由器/OpenWrt/服务器等环境
功能特性：支持自动登录校园网（eportal），可结合 cron 实现定时或开机自动执行，模拟浏览器请求稳定性较高
适用于 OpenWrt / ImmortalWrt / Linux / Windows 等环境。
环境要求：需要 Python 3.x 和 requests 库，可通过 pip install requests 安装。
